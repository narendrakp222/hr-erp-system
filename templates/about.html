About HR ERP System

The HR ERP System is a web-based application designed to manage employee information and HR operations in a centralized and efficient manner.

This project allows administrators to perform core HR tasks such as employee registration, record management, searching, updating, and deletion of employee data.

The system is developed using Flask (Python) for the backend and MySQL for data storage, with a simple and user-friendly interface built using HTML and CSS.

This HR ERP system is created as an academic project to demonstrate full-stack web development concepts including CRUD operations, session management, and template modularization.

 

  • KEY FEATURES

    • Admin Login
    • Employee Registration
    • Employee Profile Management
    • Search & Filter Employees
    • Secure Session Handling

  • TECH STACK

    • Flask (Python)
    • MySQL Database
    • HTML & CSS

  • PROJECT TYPE

    Academic / Learning Project
