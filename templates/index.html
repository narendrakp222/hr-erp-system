Welcome to HR ERP Management System

This HR ERP system is designed to help organizations manage employee records, salaries, and administrative operations efficiently using a centralized platform.

The system is developed using Flask and MySQL with secure session-based authentication for administrators.

 
 

  • ADMIN PANEL

    Only authorized HR administrators can access the system to manage employees and organizational data.

  • SYSTEM FEATURES

    • Employee Registration
    • View Employee Records
    • Search Employee
    • Session-based Login
    • Secure Logout

  • TECH STACK

    • Flask (Python)
    • MySQL Database
    • HTML & CSS
    • Session Management
