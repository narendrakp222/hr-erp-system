# HR ERP System

A full-stack HR ERP web application built using Flask, MySQL, HTML, and CSS.

## Features
- Employee Management
- HR Dashboard
- Show Employee ,Add Employee ,Search Employee and Desigination
- Payroll Management 
- Role-based Authentication

## Tech Stack
- Frontend: HTML, CSS
- Backend: Flask (Python)
- Database: MySQL (XAMPP)

## Setup Instructions
1. Clone the repository
2. Create virtual environment
3. Install dependencies
4. Import MySQL database
5. Run Flask server
